# 🌟 Adham Rakha — Premium Interactive Portfolio

Welcome to the official repository of my personal portfolio. This website is a premium, high-end interactive digital space designed to showcase my capabilities, stack, and production-ready applications. 

Designed with rich aesthetics, modern typography, glassmorphism, fluid interactive canvas simulations, and smooth 3D tilt micro-animations.

---

## 🚀 Key Features

*   **⚡ High-Performance:** Built on top of **React + Vite** for blazing-fast page loads and optimized asset delivery.
*   **🌐 Bilingual Support (EN / AR):** Dynamic real-time translation toggling with native RTL (Right-to-Left) styling.
*   **🎨 Premium Dark Theme Aesthetics:** Curated gold accents (`#B8963E`), deep slate surfaces, and glassmorphic navigation.
*   **✨ Interactive Animations:**
    *   3D Parallax Tilt effects on project cards and hero banner profile.
    *   Interactive Liquid/Fluid background canvas.
    *   Custom terminal command emulator widget.
    *   Smooth SVG-based scroll progress indicator.
*   **📱 Fully Responsive:** Adaptive layouts optimized for desktops, tablets, and mobile screens.

## 💻 Getting Started Locally

Follow these steps to run the portfolio on your local machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/earth-lion/Portfolio-Adham-Rakha.git
   cd Portfolio-Adham-Rakha
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
